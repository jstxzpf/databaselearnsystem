# SQL 简单查询详解

## 什么是SQL简单查询？

**SQL（Structured Query Language）** 是一种用于管理关系型数据库的标准语言。它由多种子句构成，其中“SELECT”语句是用来从一个或多个表中获取数据的基本语法。

### 定义和原理
简单的SQL查询使用 `SELECT` 关键字来指定要检索的数据列，并用 `FROM` 关键字来指出这些数据来自哪个表格。这样的查询通常不包含复杂的条件筛选，仅限于基本的过滤功能。
```sql
SELECT column_name FROM table_name;
```

### 在数据库系统中的作用
在关系型数据库中，SQL简单查询的主要目的是从存储的数据集中提取出用户想要查看的信息。通过这种方式，开发者或者普通用户可以快速地找到所需的数据记录，而不需要手动翻阅大量的数据文件。

### 实际应用场景和示例

#### 场景一：查找学生信息表中的所有学号
假设我们有一个包含多个学生的表格 `students`，它有三个字段：
- 学号（student_id）
- 姓名（name）
- 年龄（age）

我们要做的就是列出所有的学号。

```sql
SELECT student_id FROM students;
```

#### 场景二：查找特定科目成绩表中某科目的分数
假设我们有一个课程评分的表格 `grades`，它有三个字段：
- 学生ID（student_id）
- 科目名称（subject_name）
- 分数（score）

现在我们要找到所有学生在“数学”这门课上的分数。

```sql
SELECT score FROM grades WHERE subject_name = 'Math';
```

### SQL代码示例

为了更清晰地说明SQL简单查询的应用，我们通过一个具体例子来看如何获取数据：

**例子：**
假设有一个商店销售记录表 `sales`，其中包含：
- 销售日期（sale_date）
- 产品ID（product_id）
- 客户名（customer_name）
- 销售金额（amount）

现在我们需要列出今年所有销售额度。

```sql
SELECT sale_date, customer_name, amount FROM sales WHERE YEAR(sale_date) = YEAR(CURDATE());
```
